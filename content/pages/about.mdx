## About me

## About this blog

I am using this blog to document and share my knowledge and experience in software development. I will be writing about the following topics:
Web development, game development, devops, and more.
This blog is built with [Shadcn's blog template](https://vercel.com/templates/next.js/nextjs-contentlayer). It is built with Next.js, Markdown, and TailwindCSS.
